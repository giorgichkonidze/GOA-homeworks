useEffect არის React Hook.

იგი გამოიყენება side effects-ისთვის.

useEffect ვიყენებთ მაშინ როცა:
  1. გვინდა კოდი შესრულდეს კომპონენტის ჩატვირთვისას (mount)
  2. გვინდა კოდი შესრულდეს კონკრეტული state-ის შეცვლისას
  3. გვინდა გავაკეთოთ API
  4. გვინდა დავამატოთ event listener
  5. გვინდა გავასუფთაოთ რესურსები (cleanup)

დადებითი მხარეები:

+ საშუალებას გვაძლევს ვმართოთ side effects
+ შეგვიძლია ვაკონტროლოთ როდის გაეშვება (dependency array)
+ შეგვიძლია გავასუფთაოთ რესურსები (cleanup function)

უარყოფითი მხარეები:

- შეიძლება გამოიწვიოს ზედმეტი რენდერები
